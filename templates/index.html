RESTCONF

RESTCONF es un protocolo de gestión de dispositivos de red que utiliza la arquitectura REST (Representational State Transfer) para proporcionar una interfaz de programación de aplicaciones (API) estándar y basada en web para acceder a dispositivos de red. RESTCONF se basa en el Protocolo de configuración simple de red (NETCONF), que utiliza un formato de codificación de datos en XML. Sin embargo, a diferencia de NETCONF, RESTCONF utiliza el formato de codificación de datos JavaScript Object Notation (JSON), que es más ligero y fácil de procesar por las aplicaciones. RESTCONF también utiliza los métodos HTTP (GET, PUT, POST, DELETE) para interactuar con los recursos de los dispositivos de red.
